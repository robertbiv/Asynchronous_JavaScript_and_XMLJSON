# Asynchronous JavaScript and XML/JSON

A collection of web applications demonstrating asynchronous JavaScript programming with XML/JSON data handling, AJAX requests, and interactive DOM manipulation.

## Overview

This repository contains several JavaScript projects showcasing modern web development techniques including asynchronous data fetching, dynamic content rendering, and API integration.

## Projects

### Population Chart
Interactive data visualization displaying population statistics with dynamic chart rendering.

### TicTacToe
Classic Tic-Tac-Toe game implementation with JavaScript game logic and DOM manipulation.

### Wordcloud
Word cloud generator creating visual representations of text data with size-based word frequency display.

### Imgur Integration
Applications demonstrating Imgur API integration for image hosting and retrieval:
- **imgur**: Standard Imgur API implementation
- **imgurWeirdVersion**: Alternative implementation with different API approaches

## Technologies Used

- **JavaScript (73.1%)**: Core application logic and asynchronous programming
- **HTML (26.9%)**: Page structure and content
- AJAX/Fetch API for asynchronous data requests
- DOM manipulation for dynamic content updates
- JSON/XML data parsing and handling

## Features

- Asynchronous data fetching without page reloads
- Dynamic DOM updates based on API responses
- RESTful API integration
- Interactive user interfaces
- Real-time data visualization
- Error handling for network requests

## How to Use

1. Clone or download this repository
2. Navigate to any project folder
3. Open the `index.html` file in a modern web browser
4. For projects requiring API keys (Imgur), you may need to add your own API credentials

No build process required - projects use vanilla JavaScript.
